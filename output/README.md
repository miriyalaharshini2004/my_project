# Review Scraper - Sample Output

This directory contains sample output files demonstrating the JSON structure generated by the Review Scraper tool.

## Sample Files

- `sample_reviews_*.json`: Example output files showing the review data structure
- Generated by running `python test_scraper.py`

## JSON Structure

Each review object contains the following fields:

```json
{
  "title": "Review title or headline",
  "description": "Full review text content",
  "date": "YYYY-MM-DD format",
  "reviewer_name": "Name of the reviewer (or 'Anonymous')",
  "rating": "Numeric rating (usually 1-5)",
  "helpful_votes": "Number of helpful votes",
  "source": "G2, Capterra, or Software Advice"
}
```

## Usage

To generate sample output:
```bash
python test_scraper.py
```

To run actual scraping:
```bash
python review_scraper.py -c "Company Name" -s "2023-01-01" -e "2023-12-31" --verbose
```
